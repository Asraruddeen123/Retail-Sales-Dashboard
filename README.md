# Retail Sales & Profit Analysis Dashboard

## 📊 Project Overview
This project analyzes retail sales data using Power BI, SQL, and Excel.

## 🔧 Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard)

## 📈 Key Insights
- West region has highest sales
- Technology category generates highest profit
- High discounts reduce profitability
